# PyBot

PyBot is a GUI application containing many awesome apps made to run on the Raspberry Pi with an attached touch screen(320x240).

## Applications

* Share Market
* Twitter
* Snake Game
* Media Player
* Alarm App
* Internet Radio
* Weather App
